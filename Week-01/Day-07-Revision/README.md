# 📝 Key Concepts I Remember

✔ Software Testing improves software quality.

✔ SDLC is the complete software development process.

✔ STLC is the software testing process and is a part of SDLC.

✔ Testing Types explain **how** software is tested.

✔ Testing Levels explain **at which stage** software is tested.

✔ Smoke Testing checks whether the build is stable.

✔ Sanity Testing verifies a specific bug fix.

✔ Regression Testing checks whether existing functionality still works after changes.

✔ Retesting verifies that a reported bug has been fixed.

✔ Functional Testing validates business requirements.

✔ Non-Functional Testing evaluates performance, security, reliability, and usability.

✔ Unit Testing is performed by Developers.

✔ System Testing is performed by the QA Team.

✔ Acceptance Testing is performed by Clients or End Users.
